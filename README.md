# Presence 1

Presence 1 is a mmWave radar–based presence sensor featuring an ambient light sensor, NeoPixel RGB LED, and capacitive touch switch.  
It is designed for seamless integration with Home Assistant.

Build commands:
Python version 3.12
Esphome version 2026.1.2
Example: esphome -s device_id 3 run prd/presence_1_basic/presence_1_basic.yaml
